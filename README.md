## 字帖生成器 - 可打印 A4 米字格/田字格/方格

**本项目由阿里云ESA提供加速、计算和保护。**

> 说明：本项目用于参加“阿里云边缘安全加速ESA · Pages 边缘应用开发大赛”，提供一个可直接部署使用的字帖生成工具。

---

### 一、项目介绍

`字帖生成器` 是一个基于 **Vue 3 + Vite** 的纯前端工具：输入文字，选择字格样式和每页行列、字号、描红透明度，即可自动分页生成可打印的 A4 字帖，并支持浏览器“一键打印/保存为 PDF”。

- 支持 **米字格 / 田字格 / 方格 / 空白边框**
- 支持 **自动分页**（按每页行列数）
- 支持 **描红透明度**（适合临摹）
- 支持 **导出 HTML**（便于分享/离线保存）
- 数据默认 **本地存储**（刷新不丢）

---

### 二、使用方式

1. 输入要练习的文字（支持换行）
2. 选择字格样式、每页行列、字号、描红透明度
3. 点击「打印 / 保存为 PDF」
4. 在打印设置中选择 A4，关闭页眉页脚，边距选择默认/无，即可得到干净的字帖

---

### 三、技术说明

- **前端**：Vue 3（组合式 API）+ Vite
- **分页逻辑**：按列×行计算每页容量，换行会补齐到下一行开始
- **渲染**：CSS Grid + 伪元素绘制字格线（米字格含对角线）
- **部署**：纯静态站点，适配 ESA Pages 边缘加速与缓存

---

### 四、本地运行与构建

```bash
cd copybook-generator
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

产物在 `dist/`，可直接用于 ESA Pages 部署。

---

### 五、在阿里云 ESA Pages 上部署

1. 推送到 GitHub 公共仓库
2. 在 ESA Pages 创建站点并导入仓库
3. 构建配置：
   - **构建命令**：`npm install && npm run build`
   - **输出目录**：`dist`
4. 部署完成后获得访问 URL
